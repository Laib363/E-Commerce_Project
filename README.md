 Full-Stack E-Commerce Website

 A **fully functional E-Commerce Website** built using **Node.js, Express.js, and EJS**. It serves as a comprehensive demonstration of a full-stack application, featuring secure user authentication, authorization, and integrated image management via Cloudinary.

### Full-Stack

* **Node.js & Express.js:** Server environment and web framework.
* **MongoDB & Mongoose:** Database and ODM (Object Document Mapping).
* **EJS (Embedded JavaScript):** Templating engine for dynamic HTML rendering on the server side.
* **Authentication:** Secure **user authentication** using **Passport.js** and **Express Sessions**.
* **Cloudinary SDK:** For image storage and retrieval.
